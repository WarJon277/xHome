# Поддержка видеоформатов

## Поддерживаемые форматы и кодеки

### Рекомендуемые форматы (полная поддержка)
- **MP4** с кодеком **H.264** - лучшая совместимость с браузерами
- **WebM** с кодеками VP8/VP9 - хорошая поддержка в Chrome/Firefox
- **Ogg** с кодеком Theora - поддержка в старых версиях Firefox

### Частично поддерживаемые форматы
- **AVI** - поддержка зависит от внутреннего кодека
  - XviD, DivX - НЕ поддерживаются браузерами
  - H.264 - может работать в некоторых браузерах
- **MKV** - поддержка зависит от кодека
- **MOV** - частичная поддержка

### Неподдерживаемые форматы
- XviD, DivX (вне MP4 контейнера)
- Все форматы с нелицензированными кодеками

## Рекомендации для пользователей

### Для лучшего воспроизведения видео:
1. Конвертируйте видео в формат **MP4** с кодеком **H.264**
2. Используйте битрейт не более 2-3 Мбит/с для потокового воспроизведения
3. Разрешение видео не должно превышать 1080p для лучшей совместимости

### Рекомендуемые инструменты для конвертации:
- **FFmpeg** - командная строка
- **HandBrake** - графический интерфейс
- **VLC Media Player** - также может конвертировать файлы

### Пример команды FFmpeg для конвертации:
```bash
ffmpeg -i input.avi -c:v libx264 -preset medium -crf 23 -c:a aac -b:a 128k output.mp4
```

## Внешние плееры

Если видео не воспроизводится в браузере, используйте внешние плееры:
- **VLC Media Player** - лучшая поддержка форматов
- **MPC-HC** - для Windows
- **PotPlayer** - многофункциональный плеер
- **MPV** - легковесный плеер с отличной поддержкой форматов

## Техническая информация

Браузеры ограничивают поддержку видеоформатов по следующим причинам:
- Лицензионные ограничения
- Безопасность (кодеки запускаются в песочнице)
- Стандартизация (не все кодеки включены в стандарт HTML5)

Video.js используется как надстройка над HTML5 video, но не может обойти ограничения браузера на уровне кодеков.
